6 Performance, Security, and Constraints


6.1 Secure Login Procedures


Employees will be provided with an employee number and an initial password, as well as the ability to recover/reset passwords as needed. Role-based access control (RBAC) implementation is pertinent, limiting user access to the required features and information per their job roles and authorization. The password resets every month. Below are the login credentials that each actor must have in order to be authenticated.


Actors Authentications
Employee Login: username and password
HR Department Login: username and password
Finance Department Login: username and password
IT Department Login: username and password

6.2 Product Exclusions


6.2.1 Performance Management

Performance reviews, goal setting, and feedback are typically handled by separate performance management software


6.2.2 Advanced Reporting and Analytics

In-depth data analysis for strategic decision-making might be excluded from the core system. Integration with a dedicated Business Intelligence (BI) tool can allow for more sophisticated analysis of payroll data


6.2.3 Advanced Mobile Access

Basic mobile access for viewing payslips or requesting time off might be included, but more advanced functionalities like managing complex leave requests or expense reports could be offered as an add-on feature.
